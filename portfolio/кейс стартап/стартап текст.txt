Название кейса
Разработка маркетинговой стратегии и контент-плана для IT-продукта Structo
Описание:
structo- IT российский веб-сайт для преобразования текстовой информации в трёхмерные объекты.
задачи проекта 
Создание интуитивно понятного интерфейса для удобного создания моделей.
Мгновенная генерация 3D-моделей по текстовым запросам.
Возможность выгружать созданные модели для дальнейшего использования.
Высокая точность в создании моделей на основе пользовательских запросов.
цель кейса:
Увеличить узнаваемость бренда Structo среди ЦА на 30% за 3 месяца
Увеличить количество регистраций на платформе на 20%
Обеспечить охват в Instagram — не менее 50 000 за первый месяц
Обучить не менее 500 пользователей через обучающие гайды и посты
утп 
structo — российский сервис, который позволяет легко и быстро преобразовать текст в 3D-модель для любых задач
ЦА: 
Студенты и преподаватели вузов. Использование платформы structo для выполнения учебных заданий, создания игр, визуализации проектов и других образовательных целей. Программа поможет сэкономить время и избежать детального изучения программ по созданию 3Д моделей. 
Дизайнеры, художники и разработчики игр. Применяют structo для создания 3D-моделей, необходимых в их профессиональной деятельности, таких как разработка игр, анимация и визуализация идей. structo ускоряет процесс, позволяя быстрее получать результат.
Инженеры и архитекторы. Используют платформу для визуальной демонстрации своих проектов, моделирования и презентации инженерных и архитектурных решений.
Контент-маркетинг:
контент - маркетинг (цель контент стратегии, каналы продвижения, типы и форматы контента) 
цель контент стратегии: 
Повышение лояльности пользователей, необходимо создать долгосрочные отношения с пользователями, стимулируя их к регулярному использованию платформы. С помощью использования: публикаций о кейсах пользователей, их отзывы и акценты на функциональности платформы. 
Создание положительного имиджа и повышения доверия у пользователей платформы. С помощью использования: партнерских отношений, а также советы по применению платформы для различных профессиональных сфер. 
Повысить грамотность целевой аудитории в использовании платформы. С помощью использования: гайдов, чек-листов и инструкций по использования платформы. 
примеры: 
Образовательный контент:
Гайды: «Как создать 3D-модель из текста за 1 минуту» (пост-карусель + видео).
Чек-листы: «5 способов использовать Modelit для учебы / работы / творчества» (в формате инфографики).
Развлекательный контент:
Мемы и тренды (связанные с 3D-графикой, AI, технологическими инновациями).
Вовлекающий контент: голосования («Какой 3D-объект получился лучше?»), челленджи (создай свою первую модель и поделись результатом).
Кейсы и отзывы:
Примеры моделей, созданных на платформе (до/после, видео-демонстрации).
Истории пользователей: как structo помог в учебе/работе.

Конкуренты:
Meshy - Free 3D AI Model Generator
  structo является Российским аналогом зарубежных сервисов
  Meshy имеет систему подписки - бесплатно можно сгенерировать только несколько раз. modelit предлагает систему ограниченного количества генераций без подписки
CSM 3D Viewer
  CSM может преобразовывать только изображения, structo работает с текстом
  C structo можно выгружать объекты в качестве мэшей
Luma
  structo является Российским аналогом
  Luma перестала предлагать услуги генерации объектов и сменила вектор на генерацию видео
В целом, одно из значимых отличий structo от конкурентов — генерация 3D-моделей из текста без необходимости работы с изображениями или сложными программами. А в отличие от других зарубежных сервисов, structo доступен на русском языке.
